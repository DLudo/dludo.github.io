LOSION

In order to access the JavaScript object which provides the SoundCloud Widget API, add this script to your html page. This script exposes the SCWidget(/*iframeElement|iframeElementID*/) function to the global scope. It allows you to control the widget from the parent page (the page the widget is inserted into). SC.Widget accepts the reference to the iframe element or its id.
